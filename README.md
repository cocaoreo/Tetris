# Tetris Toy Project
## Usage
### 1. make an out file with command
  ```bash
  make
  ```
### 2. start program
  ```bash
  ./a.out
  ```

## How to Play
### 1. play
  just play tetris with recommendation block

### 2. rank
  you can check the scoreboard

### 3. recommended play
  Automatically play recomended mode

### 4. exit
